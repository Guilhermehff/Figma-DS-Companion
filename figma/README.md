# Figma Companion Workspace

Use this folder as the durable memory layer for Figma MCP work.

- `brands/`: per-brand manifests and staged brand artifacts
  Generated helper inventory: `brands/font-inventory.yml`
  Shareable helper directory: `brands/font-directory.md`
- `config/`: naming rules and taxonomy
- `decisions/`: design-system decisions
- `docs/`: stable reference docs
- `exports/`: optional dated local exports and manual snapshot references
- `history/`: dated artifacts that are no longer current source files
- `templates/`: templates for repeatable outputs
